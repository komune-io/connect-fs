


### Command

Type: `FileUploadCommand`

---

***path*** [`FilePath`](#filepath)

Path of the file to upload.



---

***metadata*** `Map<String, String>`

Additional metadata of the file



---

***vectorize*** `Boolean`

If true, will vectorize and upload the content and given metadata of the file to a vector store.<br/> /!\ The vector store API url needs to be configured for this to work.
