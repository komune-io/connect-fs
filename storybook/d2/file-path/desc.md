

## FilePath



Describe a path to a file within S3





  
<article>

***directory*** `String` 

Directory name.



(must not contain any '/')

</article>
<article>

***name*** `String` 

File name.

</article>
<article>

***objectId*** `String` 

Identifier of the object the file is attached to.



(must not contain any '/')

</article>
<article>

***objectType*** `String` 

Type of object the file is attached to.



(must not contain any '/')

</article>
