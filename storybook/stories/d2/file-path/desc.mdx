


## FilePath



Describe a path to a file within S3



---

***objectType*** `String`

Type of object the file is attached to.



(must not contain any '/')



---

***objectId*** `String`

Identifier of the object the file is attached to.



(must not contain any '/')



---

***directory*** `String`

Directory name.



(must not contain any '/')



---

***name*** `String`

File name.
